Privacy Policy

Last updated: 2025-09-05

Infinite Tool is designed to run entirely in your browser. Your files are processed on-device and are not uploaded to any server.

What we process

  • Files you select are handled locally in your browser memory for the purpose of the chosen tool (e.g., convert, compress, edit).
  • Generated results are provided as downloads via temporary blob: URLs that are revoked after use.

What we don’t collect

  • No file contents are sent to our servers.
  • No personal data is required to use the tools.

Third‑party libraries

We load some open‑source libraries from trusted CDNs with Subresource Integrity (SRI). These libraries run in your browser to enable certain features (e.g., PDF rendering).

Contact

Questions? Contact the site owner via the project repository.
